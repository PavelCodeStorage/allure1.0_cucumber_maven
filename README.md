AllureCucumberMaven
---
Overview:
---
 
Repository provide configuration example of Allure1.0 with Maven.

---

1. Execute **docker-compose up**
2. Execute **mvn clean test**
3. To perform report generation use **mvn site**
4. Open **allure-maven-plugin.html**  from target\site folder
